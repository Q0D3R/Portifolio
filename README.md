### Prerequisites

- Node.js and npm (for SCSS compilation, optional)
- Git

### Clone the Repository

```bash
git clone https://github.com/C0D3-G33K/portfolio-website.git
cd portfolio-website
```

### Install Dependencies (if using SCSS with npm scripts)

```bash
npm install
```

### Build SCSS

If you are using npm scripts or a task runner:

```bash
npm run build:scss
```

Or watch for changes:

```bash

```

Alternatively, you can compile SCSS manually or use a VSCode extension.

### Run the Project

Simply open `index.html` in your browser.
